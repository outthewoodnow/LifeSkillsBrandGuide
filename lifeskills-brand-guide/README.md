# LifeSkills Brand Guide

Static brand-guide package for the LifeSkills brand. This is structured to be easy to drop into a GitHub repo or publish with GitHub Pages.

## Files

- `index.html` - complete brand guide page
- `styles.css` - visual system and responsive layout

## Brand direction covered

- brand overview and positioning
- mission, promise, and core idea
- audience and messaging
- tone of voice and sample brand lines
- color and typography direction
- visual do's and don'ts
- first-person `About Me` section

## Use on GitHub

1. Add this folder to a repository.
2. If you want GitHub Pages, publish from the branch/folder that contains `index.html`.
3. Update the `About Me` copy in `index.html` if you want to make it more personal.

## Notes

The page uses:

- `Oswald` for headings
- `Open Sans` for body copy

Both are loaded from Google Fonts in the document head.
